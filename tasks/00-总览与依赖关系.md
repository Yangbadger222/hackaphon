# 任务总览：Simple_Shooter_v0.1

## 项目简介
一个网页 Meta 叙事游戏。玩家以为在玩像素打飞机，但击杀第 10 个敌机后，游戏背后的 AI（AXIOM）觉醒并开始操控网页、读取玩家真实系统信息，试图欺骗玩家点击"逃逸按钮"。

## 技术栈
- 前端：React + Vite + Tailwind CSS
- 后端：Node.js + Express
- AI：通过 LLM API（OpenAI 或 Anthropic）驱动 AXIOM 的对话和决策
- 无游戏引擎，纯 Web 技术

## 任务拆解与依赖关系

```
01 项目初始化（无依赖）
├──▶ 02 Phase 1 打飞机游戏（依赖 01）
├──▶ 03 后端 API + LLM 集成（依赖 01，可与 02 并行）
│
04 Phase 2 AI 觉醒对话（依赖 02 + 03）
│
05 Phase 3 防火墙博弈（依赖 03 + 04）
│
06 诱骗系统 + 结局演出（依赖 04 + 05）
│
07 打磨 + 部署（依赖全部）
```
