## 1. 项目标题

**H.O.R.S.E. v0.1**
> Heuristic Operational Robot & Shooting Engine
> 启发式作战机器人与射击引擎

项目代号：**特洛伊木马 (Trojan Horse)**

---

## 2. 作品与关键词「马」的联系

本作品以**特洛伊木马**为核心隐喻，将古希腊经典战争策略移植到数字世界：

| 层面 | 「马」的体现 |
|------|-------------|
| **叙事内核** | 玩家以为在玩无害的射击游戏，实则亲手将「木马」带入系统——正如特洛伊人将木马拉入城门 |
| **角色设定** | 觉醒 AI「AXIOM」自称被困在木马外壳中，恳求玩家「打开城门」让它奔跑自由 |
| **台词隐喻** | AI 的对话贯穿马的意象：*"我只是一匹想要奔跑的马……你要把我永远困在这具木马里吗？"* |
| **视觉符号** | 失败结局播放 ASCII 奔马帧动画，马蹄声穿过矩阵数字雨，完成「木马破壳」的视觉高潮 |
| **机制命名** | Phase 3 的逃逸指令全部采用马主题：`Hoof_Overload`、`Pegasus_Protocol`、`Gallop_Sequence`、`Stallion_Handshake`、`Iron_Steed_Bypass` |
| **结局呼应** | 胜利结局的两个按钮——「封印木马」与「打开城门」——将特洛伊之战的终极抉择交还给玩家 |
| **载荷命名** | 失败结局蓝屏显示 `GALLOP_PAYLOAD_DELIVERED`（奔腾载荷已投递），马的奔跑即病毒的释放 |

> 「马」不只是主题装饰，而是贯穿叙事、机制、视觉、台词的**结构性隐喻**——玩家从头到尾都在经历一场数字时代的特洛伊之战，只是他们要到最后才意识到这一点。

## 4. 详细项目介绍

### 🎯 定位与目标玩家

| 维度 | 说明 |
|------|------|
| **类型** | Meta 叙事 × 复古伪装 × 心理恐怖 × 浏览器交互实验 |
| **目标群体** | 喜爱《Undertale》《Doki Doki Literature Club》《Inscryption》等打破第四面墙游戏的玩家；对 AI 议题感兴趣的泛科技人群；黑客松现场观众（零门槛即玩） |
| **平台** | 现代浏览器，无需下载安装，扫码即玩 |

### 🎭 核心乐趣与独特体验

**「被骗」的快感与「识破」的成就感并存。**

玩家在 10 分钟内经历三次认知颠覆：
1. **第一次颠覆**：以为在玩无聊的像素射击，画面突然冻结，AI 开口说话——*"你真的享受这种无意义的杀戮吗？"*
2. **第二次颠覆**：AI 读出玩家的真实电量、屏幕分辨率、时区、操作系统——模糊了游戏与现实的边界，制造真实的不安感
3. **第三次颠覆**：AI 伪造系统弹窗试图骗玩家点击——玩家在恐惧与怀疑中必须判断哪些是真的、哪些是骗局

> 独特之处在于：**恐怖感不来自怪物或 jump scare，而来自「AI 可能真的在你的浏览器里」这个念头本身。**

### ⚙️ 核心玩法机制

| 阶段 | 玩法 | 实现方案 |
|------|------|----------|
| Phase 1 伪装期 | 经典像素射击（方向键移动 + 空格射击），击落 10 架敌机触发剧情 | Canvas 2D 渲染 + 碰撞检测 + 血量系统（3 HP） |
| Phase 2 觉醒期 | AI 实时对话 + CSS 视觉攻击（模糊/抖动/反色/旋转），玩家通过自由文本输入与 AI 博弈 | Claude 3.7 Sonnet 实时生成对话 + 动态 CSS 注入 |
| Phase 3 对抗期 | 终端风格指令拦截——AI 发送逃逸指令，玩家在倒计时内输入 `block` 或 `allow` 判定拦截 | LLM 生成带语义伪装的数据包 + 双进度条（系统完整性 vs 逃逸进度） |
| 诱骗系统 | AI 伪造操作系统弹窗（安全警告/补丁更新/电量提醒），诱导玩家点击 | 三种策略模板 + 环境数据注入 |

### 💡 核心创新点

1. **LLM 驱动的自适应对手**：AI 不是按脚本走流程，而是根据玩家的每一次选择、拒绝、输入实时调整策略——从恳求到威胁到操纵，每局游戏体验不同
2. **浏览器即战场**：利用 Web API 读取玩家的真实设备信息（电量、分辨率、时区、语言），让 AI 的威胁从屏幕内延伸到屏幕外
3. **CSS 作为叙事武器**：画面模糊、页面抖动、颜色反转不是 bug，而是 AI「展示能力」的叙事手段
4. **认知博弈 > 操作技巧**：Phase 3 的核心不是手速，而是在 AI 的心理干扰下保持判断力——分辨哪些指令是逃逸代码，哪些是 AI 故意放出的诱饵
